# Quantum Computing — Key Points

*Fetched live from en.wikipedia.org via Nimble (nimble extract, Tier 1 static)*

- **Qubits over bits**: Quantum computers use qubits which, unlike classical bits, can exist in superposition — representing 0, 1, or any combination simultaneously, enabling massive parallel computation.
- **Entanglement**: Qubits can be entangled, meaning the state of one instantly correlates with another regardless of distance. This is a core resource for quantum algorithms and communication.
- **Quantum speedup**: For specific problems (e.g. factoring large numbers via Shor's algorithm, searching via Grover's algorithm) quantum computers achieve exponentially faster results than classical machines.
- **Current hardware**: As of 2026, leading quantum processors (Google, IBM, IonQ) operate in the 1,000–1,500 qubit range but remain error-prone; fault-tolerant quantum computing requires error correction overhead.
- **Applications**: Most promising near-term uses include cryptography (breaking RSA), drug simulation, materials science, and optimization problems in logistics and finance.
